## Complejidades del desarrollo de la página (Tuve muchos problemas)

### 1. Alineación de elementos

Fue necesario usar **Flexbox** para acomodar el texto y las imágenes en cada sección.

### 2. Ajuste de tamaños

Se tuvieron que modificar tamaños de títulos, imágenes y botones para que se parecieran a los de la página original.
Ensayé muchos márgenes y tamaños porque al principio no me daba el mismo resultado.

### 3. Espacios entre secciones

Al inicio había demasiado espacio entre algunas partes de la página.
Se solucionó ajustando las propiedades **margin** y **padding** en el CSS.

### 4. Efecto en botones

Se agregó un efecto cuando el usuario pasa el mouse sobre los botones utilizando **:hover en CSS** para cambiar el color.

### Nota

Por el tiempo no alcancé a acomodar las imágenes en una carpeta, por lo que los archivos quedan desordenados.
